Part 2: Python Dependencies (Standardized)

Create a virtual environment (mandatory):


python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

Install Python libs:
pip install llama-cpp-python
pip install pydantic
pip install rich

Why these?
  llama-cpp-python → LLaMA bindings
  pydantic → structured outputs later
  rich → readable console output (morale matters)
